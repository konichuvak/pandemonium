Empowerment¶

class pandemonium.cumulants.Empowerment¶

Bases: pandemonium.cumulants.Cumulant

Measures the amount of causal influence an agent has on the world

Computed as a \(log(S)\), i.e. logarithm of number of reachable (in fixed number of steps) states after performing action A.

See Salge et al. 2014 for details.
