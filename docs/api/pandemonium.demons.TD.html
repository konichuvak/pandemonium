TD¶

class pandemonium.demons.TD(feature, trace_decay: float = 0, eligibility: Type[pandemonium.traces.EligibilityTrace] = <class 'pandemonium.traces.AccumulatingTrace'>, *args, **kwargs)¶

Bases: pandemonium.demons.td.TemporalDifference, pandemonium.demons.demon.PredictionDemon

Semi-gradient \(TD(\lambda)\) rule for estimating \(\tilde{v} ≈ v_{\pi}\)

\[\begin{split}\begin{align*} e_t &= γ_t λ e_{t-1} + \nabla \tilde{v}(x_t) \\ w_{t+1} &= w_t + \alpha (z_t + γ_{t+1} \tilde{v}(x_{t+1}) - \tilde{v}(x_t))e_t \end{align*}\end{split}\]

In case when \(\lambda=0\) we recover one-step \(TD(0)\) algorithm with \(e_t = \nabla \tilde{v}(x_t)\).
