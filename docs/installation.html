Installing pandemoniumÂ¶

pandemonium requires Python 3.6+ and is available on MacOS and Linux.

To begin, clone the repo from github.

git clone https://github.com/konichuvak/pandemonium.git

Note

While technically optional, we highly recommend that you use virtualenv to create an isolated python environment in which to install pandemonium.

Installing pandemonium is fairly straightforward using pip package manager.

pip install |project_name|  # use -e flag if you are a developer
